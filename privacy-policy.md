# Privacy Policy — Ghost Post

Ghost Post reads the current job posting page to extract the company name, job title, and posting date from the visible DOM. This data is sent to third-party APIs (People Data Labs, Wayback Machine, Hacker News Algolia, Airtable) solely to generate a ghost job likelihood score. Ghost Post has no backend server — all processing happens in your browser and the listed APIs. API keys are provided by you, stored locally via Chrome Sync, and never transmitted to us. No data is sold, shared, or used for advertising.
